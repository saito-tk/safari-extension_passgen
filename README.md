# パスワードジェネレータ

Safari 向けのパスワードジェネレータ拡張です。ポップアップ内で条件を指定し、複数件のパスワードを生成できます。

## 主な機能

- `crypto.getRandomValues` を使った乱数生成
- 英字(大文字) / 英字(小文字) / 数字 / 記号の個別選択
- 記号の個別選択
- 記号文字列の貼り付けから記号選択状態へ反映
- 似た文字の除外
- 同一文字の連続禁止
- 生成件数の逐次表示と進捗表示
- 長いパスワードの省略表示
  - 一覧では先頭 100 文字まで表示
  - 実際の文字数は注記で案内
- 行ごとのコピーアイコン
- テーマ切り替え
  - 青 / 緑 / ピンク / 赤 / 黄色 / オレンジ / 紫
- `storage.local` による設定保存

## UI 構成

- `設定` タブ
  - 文字種の選択
  - 記号の選択
  - 生成ルールの設定
  - 表示テーマの切り替え
- `結果` タブ
  - 生成されたパスワード一覧
  - 進捗表示 `(n/件数)`
  - コピー操作

## 入力仕様

- 文字数
  - 最小 `4`
  - 最大 `999,999`
- 件数
  - 最小 `1`
  - 最大 `30`
  - ただし文字数に応じて上限が下がります
  - 実際の上限は `min(30, floor(999999 / 文字数))`
  - 例: 文字数 `999,999` の場合、生成件数は最大 `1`

数値入力は編集中には補正せず、フォーカスが外れた時点で範囲内へ補正されます。

## 記号貼り付け機能

記号欄の下にある入力欄へ文字列を貼り付けて `反映` を押すと、対応する記号だけが選択状態になります。

この機能は、サービスごとに「使用できる記号」が異なるケースを想定しています。サービス側が利用可能な記号一覧を表示している場合、その文字列をコピーしてすぐに拡張側へ反映できるため、記号設定を手で 1 つずつ合わせる手間を減らせます。

- 対応する記号が含まれている場合
  - 含まれる記号を選択
  - 含まれない記号を非選択
- 対応外の文字しか含まれていない場合
  - 記号選択状態は変更しない
- 反映後
  - 入力欄は空になる
  - `反映` ボタンは非活性になる

## 生成中の挙動

- `生成` ボタンを押した時だけ生成を開始します
- 生成中は設定タブの入力項目を非活性にします
- 結果は 1 件ずつ追加表示されます

## コピー表示

- 各結果行の右側にコピーアイコンがあります
- コピーアイコンをクリックすると、その行のパスワードがクリップボードにコピーされます

## 強度表示

生成結果には、強度の目安として以下のラベルが表示されます。

- `Basic`
- `Good`
- `Strong`
- `Very Strong`

強度ラベルは、主に文字数と使用可能な文字種の多さをもとにした目安です。あわせて、実際に生成されたパスワードの中でユニークな文字がどれくらい使われているかも加味して、同じ文字の繰り返しが多い場合は少し弱めに表示されます。

あわせて表示される `推定 xxx bits` は、使用可能な文字種の数と文字数から見積もった理論上の強度を表します。値が大きいほど、総当たりで破られにくい傾向があります。

## Safari での実行

### 1. Xcode プロジェクトを用意する

このリポジトリは Web Extension のソースです。Safari で実行するには Xcode プロジェクト化します。

例:

```bash
xcrun safari-web-extension-converter . \
  --project-location ./xcode \
  --app-name Passgen \
  --bundle-identifier com.saitotk.passgen \
  --swift \
  --macos-only
```

### 2. Xcode で実行する

1. `xcode/Passgen/Passgen.xcodeproj` を開く
2. `Passgen` スキームで `Run`
3. Safari の `設定 > Extensions` で `Passgen` を有効化する
